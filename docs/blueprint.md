# **App Name**: TableFlow

## Core Features:

- Create Table: Create a new table with a selected shape (round or square) and a specified number of seats.
- Seat Assignment: Automatically arrange seats evenly around the selected table shape.
- Order Entry: Manually add order items to each guest's seat or the common table order.
- Status Tracking: Track the status of each order item (ordered, preparing, served, canceled) and guest (all dishes served, paid).
- Data Persistence: Automatically save and restore all table data (table shape, seats, orders, statuses) if the page is reloaded or closed.
- Intuitive Interface: Streamlined, mobile-first design for quick and easy operation on a phone.

## Style Guidelines:

- Primary color: Deep Blue (#293B5F) to evoke trust and reliability.
- Background color: Light gray (#E0E0E0) for a clean and modern look.
- Accent color: Vivid orange (#FF8C42) to highlight interactive elements and order statuses.
- Body and headline font: 'PT Sans' sans-serif font for clear readability on mobile devices.
- Simple, clear icons for statuses and actions.
- Mobile-first, single-page layout with prominent buttons and clear visual hierarchy.
